E2E Testing Skill — Construction Task Tracker

This folder contains the E2E testing skill for the project. Use the skill when authoring end-to-end tests with Playwright for core user journeys across the application.

Files:
- `SKILL.md` — full skill description, guidelines, examples, and E2E scenario templates.
- `resource/e2e_test_case_checklist.md` — reusable checklist for authoring and reviewing E2E test cases.

Purpose:
- Provide a single, detailed reference for writing production-grade, maintainable E2E tests.
- Ensure consistent quality gates across happy paths, edge cases, failure handling, accessibility, and regression coverage.
