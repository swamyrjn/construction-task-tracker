Unit Testing Skill — Construction Task Tracker

This folder contains the unit-testing skill for the project. Use it when authoring or reviewing unit and component tests with Vitest and React Testing Library.

Files:
- `SKILL.md` — complete execution guide, quality bar, and reusable test templates.
- `resource/unit_test_case_checklist.md` — concise checklist for authoring and reviewing test cases.

Purpose:
- Provide a consistent, production-grade standard for test design and implementation.
- Improve determinism, maintainability, accessibility coverage, and regression confidence.
